# rng_test.cpp

## 概述
该测试文件验证 PBRT-v4 中随机数生成器 `RNG` 类的正确性。测试涵盖种子重置的可重复性、序列前进/回退功能、差值运算符、整数分布均匀性、64 位无符号整数位分布均匀性以及双精度浮点数生成。

## 测试用例列表
| 测试用例 | 说明 |
|---|---|
| TEST(RNG, Reseed) | 验证使用相同种子重新初始化 RNG 后，能生成完全相同的随机数序列（可重复性） |
| TEST(RNG, Advance) | 测试 `RNG::Advance()` 方法的正确性：前进到指定位置后生成的值与按序列生成的值一致；支持切换序列后恢复；支持随机位置访问 |
| TEST(RNG, OperatorMinus) | 测试两个 RNG 实例之间的减法运算符 `operator-`：计算两个状态之间的步数差值，验证正负差值和通过 Advance 重新同步后差值为零 |
| TEST(RNG, Int) | 生成 10000 个随机有符号整数，验证正数和负数各占约 50% 的比例，零值占比极小，确保分布均匀性 |
| TEST(RNG, Uint64) | 生成 10000 个随机 `uint64_t` 值，统计每个比特位为 1 的频率，验证所有 64 个比特位的分布均匀性（各约 50%） |
| TEST(RNG, Double) | 生成 10 个随机 `double` 值，验证其精度高于 `float`（即 `double` 值与其 `float` 转换值不同），确认双精度生成正确 |

## 依赖关系
- `pbrt/util/rng.h` — RNG 随机数生成器核心实现
- `pbrt/util/image.h` — 图像类（仅在被注释的可视化测试中使用）
