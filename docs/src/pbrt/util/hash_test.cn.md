# hash_test.cpp

## 概述
该测试文件针对 `pbrt/util/hash.h` 中的哈希函数进行测试。测试覆盖变参哈希的正确性、大规模碰撞检测，以及非对齐内存访问的哈希一致性，确保哈希函数具有良好的分布特性和鲁棒性。

## 测试用例列表
| 测试用例 | 说明 |
|---|---|
| TEST(Hash, VarArgs) | 测试变参哈希函数的一致性，验证对单个 int64_t 值调用 `Hash()` 的结果与使用 `HashBuffer()` 对相同值进行哈希的结果完全一致 |
| TEST(Hash, Collisions) | 大规模碰撞检测测试，对 0 到 10000000 的连续整数进行哈希，检查：(1) 哈希值不等于输入值；(2) 低 32 位无碰撞；(3) 高 32 位无碰撞；(4) 完整 64 位无碰撞 |
| TEST(Hash, Unaligned) | 测试非对齐内存访问的哈希一致性，将相同数据复制到 8 种不同偏移位置的缓冲区中，验证每次 `HashBuffer` 的结果与原始对齐数据的哈希值一致 |

## 依赖关系
- `pbrt/util/hash.h` - 被测试的哈希函数模块（Hash、HashBuffer）
